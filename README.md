# Contact Recorder PWA

Professionele contactregistratie app voor jeugdhulpverleners.

## Quick Start

1. **npm install**
2. **npm run dev** (local)
3. **npm run build** (production)

## Deploy naar Vercel

1. Push naar GitHub
2. Connect repo op vercel.com
3. Deploy

## Features

- ✅ Offline-capable PWA
- ✅ By Woon & Zorg design
- ✅ Cliënt management
- ✅ Contact registratie
- ✅ Claude AI integratie (voeg je API key toe in settings)
